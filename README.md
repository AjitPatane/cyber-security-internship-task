TASK 1 : FILE INTEGRITY CHECKER

Description:
A security tool developed to monitor file changes by generating
and comparing cryptographic hash values.

Concepts Covered:
- Hashing algorithms
- File system monitoring
- Data integrity verification

Technologies:
Python
hashlib, os, json

Outcome:
Detects unauthorized modification, deletion, or addition of files.


TASK 2 : WEB APPLICATION VULNERABILITY SCANNER

Description:

Technologies:
Python

Outcome:
Helps identify potential security flaws in web applications.
